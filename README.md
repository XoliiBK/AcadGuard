Project Title:

Aegis Academic – AI-Powered Academic Risk Analysis System

Description:

Aegis Academic is a lightweight AI-powered web app designed to help students understand their academic risk early and take action before it’s too late.

As a student, it’s hard to know which module actually needs attention until marks are already low. This system solves that by combining performance tracking with Gemini 3’s reasoning capabilities to turn raw marks and syllabus content into clear, practical study guidance.

Students can add courses, define modules, upload syllabi or past papers, and enter their current assessment marks. Gemini 3 then analyzes this information to:

Estimate the probability of passing each course

Detect weak topics and performance gaps

Identify at-risk modules early

Generate personalized study strategies and focus areas

Create targeted practice questions based on uploaded materials

Instead of generic advice like “study more,” the app provides specific, actionable feedback such as which topics to prioritize and how much effort to allocate each week. The goal is to act like a personal academic “risk engine” that helps students study smarter, not just harder.

Tech Stack:

Frontend: HTML, CSS (Tailwind), JavaScript

Backend: Python (Flask)

AI Engine: Google Gemini 3 API

Data Storage: Local storage / lightweight files for persistence
